# dotfiles
dotfiles and initial PC settings with ansible.
# This repository provides...
- initial settings for macOS or ubuntu.
- backup dotfiles

# Usage
## ubuntu
```
# go to home directory
cd ~
sudo apt-get install git
git clone https://github.com/mzntaka0/dotfiles.git
cd dotfiles/ubuntu
sh ./init.sh
```
